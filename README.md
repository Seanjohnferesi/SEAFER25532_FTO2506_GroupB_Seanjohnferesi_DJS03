# Podcast App

A responsive React application that fetches and displays podcasts from an API. Users can browse podcast previews, and view detailed information in a modal.

---

## Overview

This project is a podcast app built using React. The app fetches dynamic podcast data from an API and displays it in a clean, responsive grid layout. Each podcast card shows the image, title, number of seasons, genres, and the last updated date. Clicking a card opens a modal with more detailed information about the podcast.

---

## Features

- Fetches podcast data from an API.
- Displays a grid of podcast previews.
- Each preview shows:  
  - Podcast image  
  - Podcast title  
  - Number of seasons  
  - Genres  
  - Last updated date (formatted)
- Clickable podcast cards open a **modal** with:  
  - Full description  
  - Genres  
  - Seasons with individual lists
- Loading and error states handled.
- Responsive design for desktop, tablet, and mobile.
- Reusable React components for easier maintenance.
- Consistent styling and layout with CSS.
- Documented code using JSDoc for major functions.

---

## Technologies Used

- React  
- JavaScript   
- HTML / JSX  
- CSS
- API integration for dynamic podcast data  

---

## How To Use  

- Browse the podcast grid on the landing page.  
- Click on a podcast card to open the modal and see more details.
- Hover over elements for interactive effects. 
- Use the filter dropdown (UI implemented) to select genres or categories (future functionality for filtering).
